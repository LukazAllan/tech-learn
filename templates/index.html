Suporte de Fundamentos em Python

▸ Conceito

Python é uma linguagem de programação interpretada, de alto nível e orientada a objetos, com uma sintaxe simples e fácil de ler. É amplamente utilizada em diversas áreas, como desenvolvimento web, ciência de dados, machine learning, automação e computação científica.

Principais Conceitos

  • • Variaveis

    Variáveis são usadas para armazenar informações a serem referenciadas e manipuladas em um programa de computador. Elas também fornecem uma maneira de rotular dados com um nome descritivo, para que nossos programas possam ser entendidos mais claramente pelo leitor e por nós mesmos. É útil pensar nas variáveis como contêineres que contêm informações. Seu único objetivo é rotular e armazenar dados na memória. Esses dados podem ser usados em todo o seu programa.

  • • Operadores

    Os operadores são símbolos que representam operações matemáticas, lógicas e outras ações. Eles são essenciais para construir expressões que manipulam dados e controlam a execução do código.

  • • Estruturas de seleção

    São comandos que permitem executar diferentes blocos de código com base em condições.

                                
```python
# if, elif, else - estrutura básica
idade = 18

if idade < 12:
    print("Criança")
elif idade < 18:
    print("Adolescente")
else:
    print("Adulto")

# Operador ternário
status = "Aprovado" if nota >= 6 else "Reprovado"
```

  • • Estruturas de repetição

    Permitem executar um bloco de código múltiplas vezes.

                                
```python
# for - para iterar sobre sequências
for i in range(5):  # 0 a 4
    print(i)

frutas = ['maçã', 'banana', 'laranja']
for fruta in frutas:
    print(fruta)

# while - repete enquanto condição for verdadeira
contador = 0
while contador < 5:
    print(contador)
    contador += 1

# break e continue
for num in range(10):
    if num == 3:
        continue  # pula para próxima iteração
    if num == 8:
        break     # interrompe o loop
    print(num)
```

  • • Vetores e matrizes

    Em Python, usamos listas para vetores e listas de listas para matrizes.

                                
```python
# Vetor (lista)
vetor = [1, 2, 3, 4, 5]
print(vetor[0])  # Acesso ao primeiro elemento (índice 0)

# Matriz (lista de listas)
matriz = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]
print(matriz[1][2])  # Acessa linha 1, coluna 2 (valor 6)

# Usando NumPy para arrays/matrizes mais eficientes (biblioteca externa)
import numpy as np
array = np.array([1, 2, 3])
matriz_np = np.array([[1, 2], [3, 4]])
```

  • • Orientação a Objeto

    Orientação a objetos (OO) é um paradigma de programação que organiza o código em torno de objetos, que são representações de entidades do mundo real ou abstratas, que possuem características (atributos) e comportamentos (métodos).

  • • Funções

    Uma função é um bloco de código reutilizável que realiza uma tarefa específica. Ela pode receber entradas (parâmetros), processar dados e retornar uma saída.

                                
```python
# Função com retorno
def soma(a, b):
    return a + b

resultado = soma(3, 5)

# Procedimento (função sem retorno)
def saudacao(nome):
    print(f"Olá, {nome}!")

saudacao("Maria")

# Parâmetros padrão
def potencia(base, expoente=2):
    return base ** expoente

print(potencia(3))      # 9
print(potencia(3, 3))   # 27

# Funções lambda (anônimas)
quadrado = lambda x: x**2
print(quadrado(4))  # 16
```

  • • Tratamento de exceção

    Permite lidar com erros e situações excepcionais de forma controlada.

                                
```python
# Try-except básico
try:
    numero = int(input("Digite um número: "))
    resultado = 10 / numero
    print(resultado)
except ValueError:
    print("Isso não é um número válido!")
except ZeroDivisionError:
    print("Não é possível dividir por zero!")
except Exception as e:
    print(f"Ocorreu um erro: {e}")
else:
    print("Executado com sucesso!")
finally:
    print("Este bloco sempre é executado.")

# Levantando exceções
def verificar_idade(idade):
    if idade < 0:
        raise ValueError("Idade não pode ser negativa")
    return idade >= 18

try:
    verificar_idade(-5)
except ValueError as e:
    print(e)
```
